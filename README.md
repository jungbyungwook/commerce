## 작업 내용
- 무한 스크롤
- 필터링 (품절 상품)
- 검색 기능 (자동 완성 포함)
- 상품 정보 (할인율, 기존 가격 표시)

## Getting Started

```bash
yarn install
yarn start # NOTE: JSON Server와 함께 실행됩니다. front는 3000 port, server는 3001 port입니다.
```
